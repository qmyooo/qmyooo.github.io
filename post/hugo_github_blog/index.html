macOS下使用Hugo+GitHub搭建个人博客

November 10, 2019

一、安装Hugo

打开终端输入

brew install hugo

安装完成后利用hugo version检查是否成功

二、用Hugo创建个人博客

hugo new site myblog

三、下载并设置主题

Hugo主题库地址： 以Arabica主题为例，下载主题输入

git clone https://github.com/nirocfz/arabica.git themes/arabica

四、在本地启动博客

hugo server -t arabica --buildDrafts

此时可通过http://localhost:1313/ 访问本地站点 crtl+c 结束本地访问

五、创建文章

hugo new post/blog.md

此时在content/post目录下创建了名为blog.md的markdown文件，接下来可以使用markdown编辑器对blog.md进行编辑，输入文章。 在根目录通过hugo server -t arabica --buildDrafts可以进行本地预览

六、将博客部署到远端服务器

GitHub创建一个新的repository名为username.github.io，其中username必须是小写的自己的GitHub昵称。 输入

hugo --theme=arabica --baseUrl="https://username.github.io/" --buildDrafts

生成/public目录，需要将/public上传到GitHub 输入

cd public/
git init
git add .
git commit -m "first commit"
git remote add origin https://github.com/username/username.github.io.git`
git push -u origin master

输入GitHub用户名及密码 完成。
